# 移动端样式开发计划

## 目标与范围
本计划聚焦移动端（建议断点 ≤ 768px）下三块核心场景：
1. 简历列表页：默认列表化展示，信息密度高、单手可操作。
2. 经历库页：保留卡片流展示，编辑操作改为底部抽屉。
3. 经历工厂页：导航栏重排 + 底部抽屉承载经历列表 + 中央常驻 A4 预览。

---

## 一、信息架构与交互方案

### 1) 简历列表（移动端默认列表模式）
- **默认视图**：列表（非卡片瀑布），每行包含：
  - 简历名称
  - 匹配度（百分比或进度条）
  - 创建时间
  - 操作按钮（更多菜单：编辑、复制、删除、导出）
- **交互细节**：
  - 列表项主区域点击进入详情。
  - 右侧操作按钮使用 `...` 菜单避免横向拥挤。
  - 匹配度颜色分级（高/中/低）增强快速识别。

### 2) 经历库（维持卡片顺序 + 抽屉编辑）
- **展示方式**：延续卡片顺序展示（时间倒序或业务既有顺序）。
- **编辑入口**：点击“编辑”后，页面底部弹出抽屉（Bottom Sheet）。
- **抽屉能力**：
  - 文本编辑（职责、亮点、关键词）
  - 标签维护（技能标签、行业标签）
  - 保存/取消
- **交互要求**：
  - 抽屉高度支持 60% 与全屏两档。
  - 支持手势下拉关闭，并在有未保存改动时二次确认。

### 3) 经历工厂（导航 + 抽屉 + 预览重构）
- **导航栏布局（移动端）**：
  - 左侧：常驻总匹配度
  - 中间：JD 输入下拉按钮（点击弹出“输入 JD 抽屉”）
  - 右侧：`一键组装`、`导出简历`
- **内容区域**：
  - 中间常驻 A4 简历预览（主视觉区域）
  - 预览支持长按拖动模块（拖拽排序）
- **底部抽屉区域**：
  - 承载经历列表
  - 抽屉顶部右侧提供筛选按钮（类别/匹配度/来源）

### 4) 全局导航调整
- 顶部导航改为**左上角汉堡按钮**呼出侧边栏（Drawer Sidebar）。
- 侧边栏承载原导航入口，释放顶部横向空间给核心操作。

## 二、UI 组件拆分建议

- `MobileResumeListItem`
  - 字段：`title`, `matchScore`, `createdAt`, `onMore`
- `MobileBottomSheet`
  - 统一抽屉容器，支持 snap points、手势关闭、遮罩点击关闭
- `ExperienceEditorSheet`
  - 经历编辑表单，复用现有字段校验逻辑
- `FactoryTopBarMobile`
  - 总匹配度 / JD 下拉按钮 / 一键组装 / 导出
- `JDInputSheet`
  - JD 输入、粘贴、清空、确认解析
- `ExperienceFilterBar`
  - 抽屉顶部筛选器
- `A4PreviewMobile`
  - 适配移动端缩放与拖拽排序反馈
- `AppSidebarDrawer`
  - 汉堡菜单触发，统一导航

---

## 三、响应式与样式策略

- **断点建议**：
  - `<= 768px`：移动端布局（本计划主范围）
  - `769 ~ 1024px`：平板过渡态（可选）
- **样式原则**：
  - 优先纵向流布局，减少横向滚动
  - 关键操作固定可达（底部或顶部可见）
  - 控件触控区域不低于 44px
- **排版建议**：
  - 主信息 14~16px，辅助信息 12~13px
  - 列表项间距 8~12px，抽屉内区块间距 12~16px

## 四、开发分期与里程碑

### Phase 1：基础骨架（1~2 天）
- 移动端断点样式基线
- 全局导航切换为汉堡侧边栏
- 简历列表改为移动端默认列表视图

### Phase 2：抽屉体系（2~3 天）
- 通用 `MobileBottomSheet` 封装
- 经历库编辑抽屉落地
- JD 输入抽屉落地

### Phase 3：经历工厂重构（2~4 天）
- 工厂顶部导航重排（匹配度 / JD / 组装 / 导出）
- 经历列表下沉到底部抽屉 + 筛选按钮
- A4 预览移动端适配

### Phase 4：交互增强与验收（2 天）
- 长按拖动模块排序（预览区）
- 动效、手势与边界态优化
- 回归测试与性能检查

---

## 五、验收标准（DoD）

- 简历列表在移动端默认显示为列表，且每项包含名称、匹配度、创建时间、操作按钮。
- 经历库保持卡片顺序展示，点击编辑可在底部抽屉完成文本编辑并保存。
- 经历工厂导航符合“左匹配度-中JD-右操作”结构；经历列表在底部抽屉，含顶部右侧筛选。
- 中央 A4 预览可见，支持长按拖动模块排序。
- 全局导航支持左上角呼出侧边栏，无遮挡核心操作。
- iOS Safari / Android Chrome 主流机型下无严重布局错位。

---

## 六、风险与应对

- **风险 1：拖拽与页面滚动冲突**
  - 应对：长按阈值 + 拖拽期间锁定滚动。
- **风险 2：抽屉层级与键盘遮挡**
  - 应对：键盘弹起时自动抬升抽屉内容区并滚动到焦点输入框。
- **风险 3：顶部操作过多导致拥挤**
  - 应对：次级操作折叠入更多菜单，保留核心高频按钮。

## 七、建议优先级
- **P0**：导航侧边栏、简历列表移动端默认列表、经历库编辑抽屉。
- **P1**：经历工厂导航重排、JD 抽屉、底部经历抽屉 + 筛选。
- **P2**：A4 预览长按拖拽、动效与细节优化。
