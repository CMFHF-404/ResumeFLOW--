# Resume Template Design Sources

The four templates added for ResumeFLOW are original implementations in the root Vite app. They use layout ideas from mature open-source resume projects without copying external source files into this repository.

- `open-source-classic`: inspired by JSON Resume community themes, especially compact single-column templates focused on print and ATS readability.
- `timeline-blue`: inspired by JSON Resume timeline-style community themes and the local `magic-resume-inspect` timeline component.
- `photo-card`: inspired by Reactive Resume's profile-forward template direction and local avatar preview patterns.
- `photo-sidebar`: inspired by open-source two-column resume builders, with identity/contact information in a sidebar and detailed accomplishments in the main column.

Reference projects:

- Reactive Resume: https://rxresu.me/
- JSON Resume themes: https://docs.jsonresume.org/themes
- JSON Resume class theme: https://github.com/jsonresume/jsonresume-theme-class
- Local independent workspace reference: `magic-resume-inspect/src/components/templates/`
